# money-v3
## 开发前准备
手动添加本地配置文件
增加 coin/coin.yaml 
增加 config.yaml 

## 开发

```bash
npm install
```

## 运行

```bash
1.需要安装pm2 环境
2.采用pm2 守护进程 支持运行多个node程序
3.全局安装
npm install -g pm2
```
